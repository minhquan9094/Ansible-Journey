# Gathering information from host

- Using Discovering variables: facts and magic variablesïƒ

##### In Playbook

```

- name: Print all available facts
  ansible.builtin.debug:
    var: ansible_facts

```

```
## Disable fact in playbook

- name: Print information
  host: all
  garther_facts: no
  task:
    - debug:
        var: ansible_facts

```
